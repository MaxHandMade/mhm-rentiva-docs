Destek URL'leri wpalemi.com'a güncellendi, v4.40.0–v4.49.0 için changelog backfill yapıldı ve README, React SPA admin arayüzü bölümüyle güncellendi.

<!--truncate-->

## Ne Değişti

### Destek URL'leri Güncellendi

Eklenti içindeki tüm destek bağlantıları ve destek e-posta adresi güncellendi:

- Destek URL'leri artık **wpalemi.com**'a yönlendiriyor
- Destek e-postası **support@wpalemi.com** olarak güncellendi

### Changelog Backfill

**v4.40.0 ile v4.49.0** arasındaki sürümler için dokümantasyon changelog girişleri hem İngilizce hem Türkçe olarak tamamlandı — Vendor Management Faz A sprint'i, Messages Faz 3, Ödeme Talepleri, Frontend CSS izolasyonu ve mobil duyarlı düzeltmeler dahil.

### README Güncellemesi

Eklenti README'si artık taşınan admin sayfalarını belgeleyen ayrı bir **React SPA Yönetici Arayüzü** bölümü içeriyor: Kontrol Paneli, Mesajlar, Müşteriler, Raporlar, Vendor Yönetimi (Faz A + Faz B sekmeleri).

### Paket

Bu sürüm ayrıca v4.46.0 veya öncesinden güncelleyen operatörler için v4.49.0'daki mobil breakpoint düzeltmesini (768 piksel → 782 piksel) de içeriyor.
