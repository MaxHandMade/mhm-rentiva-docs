Admin arayüzü için üç mobil uyumlu CSS düzeltmesi: Mesajlar tablosu kart düzeni, KPI stat kart kılavuz daraltması ve Raporlar tarih filtresi dikey istifi. Tümü, WP admin kenar çubuğunun daraltıldığı gerçek mobil viewport'leri hedefliyor.

<!--truncate-->

## Ne Değişti

### Mesajlar Tablosu — Mobil Kart Düzeni

≤ 375 piksel viewport genişliklerinde, Mesajlar tablosu artık **tek sütunlu kart düzenine** geçiyor. Sütun başlıkları, `<td>` elementlerindeki `data-label` özelliklerini okuyan CSS `::before` sözde elementleri aracılığıyla render ediliyor — JavaScript gerektirmiyor.

### KPI Stat Kartları — Duyarlı Kılavuz

Stat kart kılavuzu artık duyarlı biçimde daralıyor:

- ≤ 768 piksel → **2 sütun**
- ≤ 480 piksel → **1 sütun**

Müşteriler, Raporlar ve Kontrol Paneli admin sayfalarına uygulanıyor.

### Raporlar Tarih Filtresi — Dikey İstif

Raporlar sayfasındaki tarih aralığı filtresi (Başlangıç Tarihi / Bitiş Tarihi) artık ≤ 768 piksel'de **dikey istife** geçiyor; mobil cihazlarda input'ların viewport'u taşmasını önlüyor.
