🔗 v4.26.8 · URL State

  • 🔴 Şu an: Kullanıcı BMW + İstanbul + Fiyat artan filtreler, linki paylaşır — karşı taraf tüm araçları görür. Filtreler URL'e hiç yansımıyor.
  • 🟢 v4.26.8: Her filtre ve sıralama değişikliği URL'e yazılır. Sayfayı yenile → aynı durum. Link paylaş → aynı sonuç.
  • 📌 replaceState: Her değişiklik yeni history entry oluşturmaz — geri tuşu çalışmaya devam eder. pushState değil; tek entry, sürekli güncellenen URL.
Örnek URL
/araclar/?brand=BMW&location=istanbul&orderby=price&order=ASC
// search-results.js — reloadResults() içinde const params = new URLSearchParams( getCurrentFilters($scope) ); history.replaceState( null, '', '?' + params.toString() );
Sayfa yüklenişinde URL params → form elemanlarına yansıtılıyor (PHP zaten yapıyor, AJAX re-render sonrası JS de yapacak).
