# Yaygın Sorunlar

## İçindekiler
- Rezervasyon kaydı oluşmuyor
- Ödeme sonrası durum güncellenmiyor
- Admin listesinde veri eksik görünüyor

## Rezervasyon kaydı oluşmuyor
- Kısa kod eşlemelerini doğrulayın.
- Form nonce ve cache durumunu kontrol edin.
- `Booking` loglarını inceleyin.

## Ödeme sonrası durum güncellenmiyor
- Webhook adresini ve imza doğrulamasını kontrol edin.
- Gateway test modunun açık olup olmadığını doğrulayın.

## Admin listesinde veri eksik görünüyor
- Filtreleri temizleyin.
- Cache sıfırlayın.
- Yetki seviyesini kontrol edin.

## Bölüm Sonu Özeti
- Yaygın Sorunlar sayfası, tekil referans başlıklarıyla standart dokümantasyon yapısına alınmıştır.
