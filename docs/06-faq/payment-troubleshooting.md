# Ödeme Sorun Giderme

## İçindekiler
- Hata 400/401
- Hata 403
- Hata 500
- Durum eşleşmesi

## Hata 400/401
- API anahtarları ve ortam (sandbox/live) uyumunu kontrol edin.

## Hata 403
- IP kısıtı, WAF veya güvenlik eklentisi engeli olabilir.

## Hata 500
- Sunucu logları ve callback endpoint hata çıktısını inceleyin.

## Durum eşleşmesi
- Gateway dönüş kodunu Rentiva durum matrisiyle karşılaştırın.

## Bölüm Sonu Özeti
- Ödeme Sorun Giderme sayfası, tekil referans başlıklarıyla standart dokümantasyon yapısına alınmıştır.
