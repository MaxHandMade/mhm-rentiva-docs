# Performans ve Önbellek

## İçindekiler
- Yavaş admin ekranları
- Yanlış/eskimiş veri
- Ölçüm

## Yavaş admin ekranları
- Liste sorgularında filtre ve sayfalama kullanın.
- Ağır raporları zamanlanmış görevle üretin.

## Yanlış/eskimiş veri
- Uygulama cache ve sayfa cache ayrımını kontrol edin.
- Kritik ayar değişimlerinde cache invalidation çalıştığını doğrulayın.

## Ölçüm
- TTFB, sorgu sayısı ve cron sürelerini izleyin.

## Bölüm Sonu Özeti
- Performans ve Önbellek sayfası, tekil referans başlıklarıyla standart dokümantasyon yapısına alınmıştır.
