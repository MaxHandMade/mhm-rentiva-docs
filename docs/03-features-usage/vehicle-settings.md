# Vehicle Settings Sayfası

`MHM Rentiva > Vehicle Settings`, araçlarınızda hangi detayların, özelliklerin ve ekipmanların kullanılacağını yapılandırdığınız alandır. Bu panel sayesinde formlarda ve araç kartlarında gereksiz alanları gizleyebilir veya kendi özel alanlarınızı ekleyebilirsiniz.

## Genel Bakış

Ayarlar sayfası üç ana bölümden oluşur:
1.  **Vehicle Details:** Aracın temel bilgileri (Fiyat, Yıl, Kilometre vb.).
2.  **Vehicle Features:** Aracın konfor ve teknik özellikleri (Klima, ABS vb.).
3.  **Vehicle Equipment:** Araçta bulunan ek donanımlar (Stepne, İlk Yardım Seti vb.).

Her bölümde, sistemde tanımlı standart alanları seçip devre dışı bırakabilir veya "Add Custom" özelliği ile sınırsız sayıda yeni alan tanımlayabilirsiniz.

## 1. Vehicle Details (Araç Detayları)

Araçların kaydedilirken sorulacak temel sorularını belirler.

| Standart Alan | Açıklama |
|---|---|
| Price/Day | Günlük kiralama bedeli. |
| Year | Aracın model yılı. |
| Mileage | Aracın mevcut kilometresi. |
| License Plate | Plaka bilgisi. |
| Color | Araç rengi. |
| Brand | Marka bilgisi. |
| Model | Model bilgisi. |
| Seats | Koltuk sayısı. |
| Doors | Kapı sayısı. |
| Transmission | Vites tipi (Manual/Automatic). |
| Fuel Type | Yakıt tipi (Petrol/Diesel/Hybrid/Electric). |
| Engine Size | Motor hacmi. |
| Availability | Aracın mevcut durumu (Active/Maintenance/Inactive). |

## 2. Vehicle Features (Özellikler)

Aracın sunduğu yerleşik özellikleri tanımlar.

- Air Conditioning
- Power Steering
- ABS Brakes
- Airbags
- Central Locking
- Electric Windows
- Power Mirrors
- Fog Lights
- Cruise Control
- Bluetooth
- USB Port
- Navigation
- Sunroof
- Leather Seats
- Heated Seats

## 3. Vehicle Equipment (Ekipmanlar)

Araçla birlikte verilen taşınabilir veya sabit donanımları içerir.

- Spare Tire (Stepne)
- Jack (Kriko)
- First Aid Kit (İlk Yardım Çantası)
- Fire Extinguisher (Yangın Tüpü)
- Warning Triangle (Reflektör)
- Jumper Cables (Akü Takviye Kablosu)
- Ice Scraper (Buz Kazıyıcı)
- Car Cover (Branda)
- Child Seat (Çocuk Koltuğu)
- GPS Tracker
- Dashcam (Araç Kamerası)
- Phone Holder (Telefon Tutucu)
- Charger (Şarj Aleti)
- Cleaning Kit (Temizlik Seti)
- Emergency Kit (Acil Durum Seti)

## Özel Alan Ekleme (Custom Fields)

Her bölümün altında bulunan **"Add Custom"** kutucuğunu kullanarak kendi tanımlarınızı yapabilirsiniz.
1.  Metin kutusuna özelliğin adını yazın (Örn: "Wifi Hotspot").
2.  İlgili **Add Custom** butonuna tıklayın.
3.  Yeni alan listenin en altına eklenecektir.
4.  Kaydettikten sonra bu yeni alan tüm araç düzenleme sayfalarında görünecektir.

:::tip
Eklediğiniz özel alanları daha sonra "Remove" butonu ile silebilirsiniz. Ancak bu alanı kullanan araçlardaki veriler de silinecektir.
:::

## Ayarları Kaydetme

Seçimlerinizi yaptıktan sonra sayfanın altındaki **"Save Settings"** butonuna basarak değişiklikleri kaydedin. Bu değişiklikler anında tüm sisteme yansır.
