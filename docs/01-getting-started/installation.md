# Kurulum ve İlk Ayarlar Rehberi

## İçindekiler
- Gereksinimler
- Kurulum Adımları
- İlk Doğrulama

## Gereksinimler
- WordPress 5.0+
- PHP 7.4+
- Kalıcı bağlantılar: `Yazı ismi`
- HTTPS (Ödeme akı için önerilir)

## Kurulum Adımları
1. `mhm-rentiva.zip` dosyasını yükleyin ve etkinleştirin.
2. `MHM Rentiva > Settings` altında temel ayarları yapın.
3. Zorunlu sayfaları ve kısa kodları eşleyin.
4. Ödeme entegrasyonunu test modunda doğrulayın.

## İlk Doğrulama
- Bir Örnek araç ekleyin.
- Örnek rezervasyon akışını tamamlayın.
- E-posta tetiklerini test edin.
