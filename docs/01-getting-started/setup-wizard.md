# Kurulum Sihirbazı

## İçindekiler
- Amaç
- Adımlar
- Notlar

## Amaç
Setup Wizard, ilk kurulum adımlarını tek akışta tamamlamanızı sağlar.

## Adımlar
1. Site ve Şirket bilgileri
2. Sayfa eşlemeleri
3. Ödeme ve e-posta başlangıç ayarları
4. Son kontrol ve kayıt

## Notlar
- İstediğiniz zaman `MHM Rentiva > Setup Wizard` üzerinden tekrar açabilirsiniz.
- Üretime çıkmadan önce test rezervasyonu önerilir.

## Bölüm Sonu Özeti
- Kurulum Sihirbazı sayfası, tekil referans başlıklarıyla standart dokümantasyon yapısına alınmıştır.
