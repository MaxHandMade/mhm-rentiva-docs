# Payout List Table

## İçindekiler
- Sorumluluk
- Kolonlar ve Filtreler

## Sorumluluk
`PayoutListTable`, payout kayıtlarının yönetici panelinde listelenmesi, filtrelenmesi ve incelenmesini sağlar.

## Kolonlar ve Filtreler
- Durum
- Vendor
- Tutar
- Tarih/sıralama alanları

## Aksiyonlar
- Kayıt detayına geçiş
- Durum tabanlı işlem aksiyonları
- CSV export akışıyla birlikte çalışma

## Performans Notları
- Sorgular filtreye göre daraltılmalıdır.
- Toplu operasyonlarda nonce/yetki kontrolleri zorunludur.

## Bölüm Sonu Özeti
- `PayoutListTable`, finans operasyonunun günlük görünürlüğünü sağlayan ana UI bileşenidir.
