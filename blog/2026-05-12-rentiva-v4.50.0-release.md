Support URLs updated to wpalemi.com, changelog backfilled for v4.40.0–v4.49.0, and README updated with the React SPA admin interface section.

<!--truncate-->

## What Changed

### Support URLs Updated

All in-plugin support links and the support email address have been updated:

- Support URLs now point to **wpalemi.com**
- Support email updated to **support@wpalemi.com**

### Changelog Backfill

Documentation changelog entries for **v4.40.0 through v4.49.0** have been backfilled in both English and Turkish — covering the Vendor Management Faz A sprint, Messages Faz 3, Payout Requests, Frontend CSS isolation, and mobile responsive fixes.

### README Update

The plugin README now includes a dedicated **React SPA Admin Interface** section documenting the migrated admin pages: Dashboard, Messages, Customers, Reports, Vendor Management (Faz A + Faz B tabs).

### Bundle

This release also bundles the mobile breakpoint fix from v4.49.0 (768 px → 782 px) for operators updating from v4.46.0 or earlier.
