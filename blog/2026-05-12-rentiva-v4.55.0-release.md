KPI cards and table action buttons on the Shortcode Pages admin screen are now accessible at narrow widths.

<!--truncate-->

## What's New

### Shortcode Pages — Mobile Accessibility

- **KPI cards:** `flex-wrap` added — all three stat cards (Total / Active / Missing) are now visible at ≤782px instead of overflowing off-screen.
- **Table:** horizontal scroll wrapper added — the Edit / View / Remove action buttons are reachable via scroll at narrow widths.
