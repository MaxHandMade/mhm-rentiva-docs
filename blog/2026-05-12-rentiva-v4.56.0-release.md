Consistent gradient KPI cards now apply across all admin list pages. Vehicles, Bookings, Transfer, and Additional Services adopt the Dashboard/Reports blue/green/amber/grey gradient style.

<!--truncate-->

## What's New

### Unified KPI Cards

- **Vehicles, Bookings, Transfer, and Additional Services** list tables now use the same gradient KPI style as Dashboard and Reports.
- The shared `shared/admin.css` classes (`.mhm-stat-card` / `.mhm-stats-grid`) were applied to the PHP-rendered pages.
- CSS variable scope was extended to `.mhm-stats-grid` so the gradients work outside React roots too.
