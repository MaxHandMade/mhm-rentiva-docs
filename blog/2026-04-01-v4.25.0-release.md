v4.25.0, vendor bildirimleri için yönetici panelinden düzenlenebilir e-posta şablonları getiriyor. 22 vendor bildirim şablonu artık Ayarlar ekranından özelleştirilebilir. Ayrıca Site Instance Check-in cron'u Cron İş Monitörü'ne eklendi ve `weekly` WP cron interval'inin güvenilir kaydı garanti altına alındı.

<!--truncate-->

## 🚀 Öne Çıkan Yenilikler

### ✉️ Yönetici Tarafından Düzenlenebilir Vendor E-posta Şablonları

22 vendor bildirim şablonu artık doğrudan admin panelinden düzenlenebilir: **Ayarlar → Bildirim Şablonları → Vendor Bildirimleri** sekmesi.

Şablonlar 4 bölüme ayrılmış durumda:

| Bölüm | Şablon Sayısı |
|-------|--------------|
| Vendor Hesap (kayıt, onay, ret, askı) | 5 |
| Araç İnceleme (onay, ret, düzenleme, yeniden inceleme) | 4 |
| Araç Yaşam Döngüsü (duraklatma, devam, sona erme vb.) | 9 |
| Finansal E-postalar (ödeme, iade, fatura) | 4 |

**Akıllı Fallback Sistemi:**
- Konu alanı boş bırakılırsa → varsayılan konu metni gösterilir (admin ne gönderileceğini görür)
- Gövde alanı boş = Gold Standard HTML dosya fallback — özelleştirme yapılmadığında profesyonel tasarım korunur

### 🌐 Çeviri Desteği

Vendor e-posta admin arayüzü için **56 yeni Türkçe çeviri** eklendi.

## 🛠️ Teknik İyileştirmeler

### 🎨 Gold Standard E-posta Şablonu Gradient Düzeltmesi

Header başlığında gerçek renk geçişi artık doğru üretiliyor — önceki sürümde gradient oluşturma hesaplama hatası vardı.

### ♿ Erişilebilirlik: CTA Buton Kontrast Oranı

Araç yeniden inceleme admin e-postasındaki amber CTA butonunun kontrast oranı **2.97:1'den 6.8:1'e** yükseltildi. WCAG AA standardını karşılıyor.

### 🔧 Site Instance Check-in Cron Görünürlüğü

`mhm_rentiva_instance_checkin` cron görevi artık **Cron İş Monitörü**'nde görünüyor ve "Şimdi Çalıştır" butonu ile manuel tetiklenebiliyor. Bu sayede Lite site instance'larının lisans sunucusuna kaydı izlenebilir.

### 🔧 Weekly Cron Interval Garantisi

`weekly` WP cron interval'i artık eklenti tarafından açıkça kayıt ediliyor — WooCommerce gibi harici eklentilere bağımlılık kaldırıldı.
